删除docker进程不再使用的image​

docker system prune -a --volumes

::: warning 当您的docker占用过多的空间时，该命令执行可能会比较耗时。 :::

通过grep过滤批量删除image​

docker images | grep 'ccr.ccs.tencentyun.com/cdb.khaos' | awk '{print $3}' | sort | uniq | xargs docker rmi
