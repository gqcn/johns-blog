### VSCode 配置

`VS Code Copilot Agent` 执行时会询问三类权限：

| 权限类型 | 配置项 | 说明 |
|----------|--------|------|
| 终端命令 | `chat.tools.terminal.autoApprove` | 控制哪些`shell`命令可自动执行 |
| URL 访问 | `chat.tools.urls.autoApprove` | 控制哪些`URL`可自动访问 |
| 文件读写 | 无专用`autoApprove`配置 | 文件改动以`diff`形式展示供审查，这是安全设计行为 |

为了避免 `AI` 不停询问权限，一些自动允许的权限设置：
