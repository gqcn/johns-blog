## 环境准备

### `Claude Code`插件

通过的`Claude Code`的插件市场方便安装`Skills/MCP`等工具（一次居然只能安装一个插件）。
- code-review
- code-simplifier
- context7
- frontend-design
- gopls-lsp
- playwright
- ralph-loop
- skill-creator

### `Claude Code`配置

```json title=".claude/settings.local.json"
{
  "permissions": {
    "allow": [],
    "defaultMode": "bypassPermissions"
  }
}
```
